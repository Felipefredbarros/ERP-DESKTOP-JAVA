# ERP-DESKTOP-JAVA
Sistema Desktop de controle de estoque com movimento venda para uma loja varejista de roupas. Feito para ampliar conhecimentos em programação em Java Swing + Banco de dados relacional

RECURSOS

-Cadastro de Clientes persistindo no banco;

-Cadastro de Funcionarios persistindo no banco;

-Cadastro de Fornecedores persistindo no banco;

-Cadastro de Produtos persistindo no banco;

-Movimentação de venda entre um cliente e um funcionario, podendo selecionar os produtos cadastrados por meio de uma carrinho de compra.

-Relatorio de Clientes cadastrados;

-Relatorio de Funcionarios cadastrados;

-Relatorio de Fornecedores cadastrados;

-Relatorio de Produtos cadastrados;

-Relatorio das Vendas realizadas;

-Relatorio de Produtos mais Vendidos;
